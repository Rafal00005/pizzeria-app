# Pizzeria App

A restaurant table management application built with React.

Live Demo: https://replit.dev/@rafalnierstreba/pizzeria-app

## What it does

- Display list of tables from server
- Edit table details (status, people count, bill)
- Automatic data validation (max 10 people)
- Save changes to server
- Nice UI with Bootstrap

## Technologies

- React
- Redux
- React Router
- Bootstrap
- JSON Server

## How to run locally
```
yarn install
yarn start
```
